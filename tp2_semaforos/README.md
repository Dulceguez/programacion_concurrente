# TP2 - Semáforos

## 📌 Descripción

Este trabajo práctico aborda el concepto de **sincronización de procesos mediante semáforos** en programación concurrente.

Los semáforos son un mecanismo utilizado para controlar el acceso a recursos compartidos, evitando condiciones de carrera y asegurando la correcta coordinación entre procesos o hilos.

---

## 🎯 Objetivo

- Comprender el funcionamiento de los semáforos.
- Aplicar mecanismos de sincronización en sistemas concurrentes.
- Resolver problemas de acceso a secciones críticas.
- Coordinar la ejecución de procesos concurrentes.

---

## 🧠 Conceptos trabajados

- Semáforos binarios y contadores  
- Exclusión mutua  
- Secciones críticas  
- Sincronización de procesos  
- Coordinación entre hilos
