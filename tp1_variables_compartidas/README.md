# TP1 - Variables Compartidas

## 📌 Descripción

Este trabajo práctico aborda el concepto de **concurrencia mediante variables compartidas**, uno de los modelos básicos de comunicación entre procesos o hilos en programación concurrente.

Se estudian los problemas que surgen cuando múltiples procesos acceden y modifican datos comunes de manera simultánea, como condiciones de carrera (race conditions), y la necesidad de sincronización.

---

## 🎯 Objetivo

- Comprender el uso de variables compartidas en sistemas concurrentes.
- Identificar problemas de sincronización y acceso concurrente.
- Analizar condiciones de carrera.
- Aplicar técnicas básicas de control de concurrencia.

---

## 🧠 Conceptos trabajados

- Concurrencia en procesos/hilos  
- Variables compartidas  
- Condiciones de carrera (race conditions)  
- Secciones críticas  
- Necesidad de sincronización  

---

## ⚠️ Problemas abordados

Cuando varios procesos acceden a una variable compartida sin control adecuado, pueden producirse resultados inconsistentes o incorrectos debido al acceso simultáneo.

---

## 💡 Solución general

Se analiza la importancia de proteger las secciones críticas mediante mecanismos de sincronización para garantizar la consistencia de los datos compartidos.
